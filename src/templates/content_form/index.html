What is this?

Create a JSON Server with Flask Framework

This application allows users to create a JSON server via a designated endpoint. This makes it easier to create mock APIs and speed up the development process.

Creating
The user inputs their endpoint, server name, and JSON content, and clicks the "Let's create" button. The application then displays the full URL of the created JSON server to the user.
Showing
When the application receives a request to the full URL created by the user's designated endpoint, it returns the specified data. This allows users to act as if they are using a real API during testing and development.
